# Mini QA System

A Mini Question-Answering system built using a pretrained Transformer model (DistilBERT / RoBERTa).
This project contains:
- `data/context.txt` — a 200+ word paragraph (Health domain) used as context.
- `src/qa_system.py` — the QA script using Hugging Face transformers.
- `requirements.txt` — dependencies.
- `.gitignore`
